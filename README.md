  // WHILE loop
  ┌────────────────────────────┐
  │ while (condition)          │
  │ {                          │
  │   // CODIGO A EJECUTAR     │
  │ }                          │
  └────────────────────────────┘

  // DO-WHILE loop
  ┌────────────────────────────┐
  │ do {                       │
  │   // CODIGO A EJECUTAR     │
  │ } while(cond);             │
  └────────────────────────────┘

+-------+     +----------+     +-------+
| START | --> | Condición | --> |  Loop | --> (Si la condición es verdadera)
+-------+     +----------+     +-------+       |
                   |                           |
                   | (Si es falsa)             |
                   V                           |
               +-------+                       |
               |  END  | <---------------------+
               +-------+

</pre>
